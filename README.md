## Usage
- 默认下载种子到 ~/Downloads/ 文件夹下
- 默认使用 ⇧⌘D 打开种子下载到文件夹
- 可在设置中调是下载种子，还是使用系统默认应用打开 magnet 链接
- 可在设置中调整下载文件夹的位置
- 可添加关键字（按 `Return` 键或者 `添加` 按钮）
- 可删除关键字（按 `Command+Delete` 或者 `Edit ➡ 删除关键字`）
- 可设置自动下载当天关键字种子间隔

个人使用方法：把每天要看动画的关键字+字幕组+字幕类型设定好之后，每隔 5 分钟让这个工具自动检测是否有新种子。
